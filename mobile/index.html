Previous Projects and Assignments

In this section, I've included many projects I've worked on throughout college. Many of them were fun experiences to me and full of programming no-no's. Nonetheless, I've learned so much from each of them.

Content Management System

One of my first major projects of college was to build a simple CMS with php. It ended up taking longer than I expected, but I had a lot more fun with it than expected too. I struggled with the CSS because I didn't have much exposure to it. That's why I went crazy with the gradients.

Orbituary

Orbituary was a game I made in Python along with a friend. It took of hard work and strange maths. My target for the game was to make it completely scalable; for example, you can set the number of enemies that will appear on a given level, and the game will evenly distribute the enemeies throughout the level with a small degree of randomness. The game didn't look great or play the best, but there are some good ideas about this project that I will remember.

Enterprise Computing Web Application

My Enterprise Computing course focused on all of the aspects demonstrated by this project. Using C#, a friend and I created a web application demonstrating all of the CRUD operations. We allowed a user to add cars, engines, and manufacturers into a database through a simple GUI. Unfortunately, this project was setup on a temporary Windows Azure server, so I do not have a link to the working application.

Monu-mental Math App

A friend and I created this simple Android app that has two game modes (Mental Math and Fleeting Figures) based around math. Using Java, we created this simple yet slightly addictive game to test your quick thinking and memorization abilities.

Coin Collector

One of the simplest yet most fun games I made. This was also made with Python. You simply steer the red car with your pointer around obstacles such as wreckage and cars (both with and oncoming traffic). Try to collect as many coins as you can before you inevitably explode (unless you're really good)! You get bonuses for certain pickups, like a star that gives you invulnerability (wonder where I got that idea from).

Slot Machine

One last Python game I made using winning percentages based on actual slot machines. This slot machine is a simple and far less risky method of gambling than your standard machine. It's also far more ugly, but it works!
